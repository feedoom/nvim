# nvim

my nvim config

## 安装依赖软件

### pip pynvim

`pip3 install pynvim`

### yarn neovim

`yarn global add neovim`

### ctags for tagbar

`sudo apt-get install ctags`

### ag for ag.vim

`apt-get install the_silver_searcher`

### xsel for clipborad

`apt-get install xsel`

### nodejs,yarn for coc.nvim

### 预览html
```
cd ~/.config/nvim/plugged/bracey.vim
npm install --prefix server
```

### leetcode
pip3 install keyring browser-cookie3


### END
安装完后应运行`:checkhealth`
## 键位
| 快捷键   | 操作               |
|----------|--------------------|
| F2       | 剪贴模式           |
| F3       | 取消搜索高亮       |
| F4       | 打开终端           |
| F5       | 运行当前文件       |
| F6       | 单词检查           |
| F7       | 格式化             |
| F8       | 打开剪贴板历史记录 |
| tg       | 打开tagbar         |
| tr       | 打开文件书         |
| un       | 打开撤销树         |
| leader a | Ag搜索             |
| leader f | leaderF搜索        |
| leader w | 翻译               |
