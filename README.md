my nvim config

## 安装依赖软件

### pip pynvim

`pip3 install pynvim`

### yarn neovim

`yarn global add neovim`

### ctags for tagbar

`sudo apt-get install ctags`

### ag for ag.vim

`apt-get install the_silver_searcher`

### xsel for clipborad

`apt-get install xsel xclip`

### nodejs,yarn for coc.nvim

### leetcode
pip3 install keyring browser-cookie3
